# Virtual Business Card

A responsive and interactive virtual business card built using HTML, CSS, and JavaScript.

## Features
- Responsive design
- Dark / Light mode
- Interactive hover effects
- Clickable contact & social links
- QR code for easy sharing
- Deployed on GitHub Pages
